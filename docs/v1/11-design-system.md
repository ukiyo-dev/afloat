# 设计系统与 UI 边界

本文档定义 Afloat v1 的前端视觉语言、交互模式与页面路由边界。

## 1. 核心视觉风格：数据账本 (Ledger) 与粗野主义 (Brutalism)

Afloat 的核心是“时间的一面镜子”。它不应带有催促感、社交属性或现代常见的“AI Slop”柔和圆角卡片风。它是一个客观记录事实与承诺的终端。

- **去装饰化**：大量使用硬边框 (`border-2 border-ink`) 和纯色实阴影 (`shadow-[4px_4px_0_0_#111]`)，摒弃渐变、圆角和半透明悬浮层。
- **高对比编辑风 (Editorial)**：排版上采用极高对比度的字号跳跃。重点数据和标题极大，次要信息极小并全大写。
- **色彩规范 (Tailwind Theme)**：
  - `paper` (`#f0ece1`): 未漂白的纸质背景色，带有网格点纹理。
  - `ink` (`#111111`): 深墨黑色，用于几乎所有文字和边框。
  - `ink-light` (`#4a4a4a`): 次要信息、时间戳。
  - `highlight` (`#d5ff00`): 荧光黄，用于高亮、状态警示和选中状态，模拟荧光笔。
  - `danger` (`#ff3333`): 警示红，用于严重错误（重叠冲突、失衡过期线程）。
  - `success` (`#00e676`): 成功绿。
  - **语义颜色映射 (Semantic Colors)**：在表达具体时间分配时，必须遵循以下硬性色彩规则：
    - `工作/理想 (Work/Ideal)`: 蓝色系 (`semantic-work`: `#0055ff`)
    - `娱乐 (Leisure)`: 深绿色系 (`semantic-leisure`: `#005c29`)
    - `休息 (Rest)`: 浅绿色系 (`semantic-rest`: `#a7f3d0`)
    - `外部偏移 (External Shift)`: 黄色系 (`semantic-ext`: `#ffb300`)
    - `内部偏移 (Internal Shift)`: 红色系 (`semantic-int`: `#ff3333`)

## 2. 排版 (Typography)

前端全局应用两套特征鲜明的字体：

- **展示字体 (Serif)**：使用 `Noto Serif SC`（宋体/明体）。仅用于页面大标题（如 `浮生`）和数据条目的强调说明。
- **数据与界面字体 (Mono)**：使用 `JetBrains Mono`。作为系统的默认字体（`font-mono`），用于一切数字、表单输入、按钮、标签（Label）、路由导航和 UI 控件。

## 3. 交互与空间管理

尽量避免引入复杂的 JavaScript 交互状态库，使用原生 HTML 特性来保持“硬核”感与服务端渲染性能。

- **原生折叠 (`<details>` / `<summary>`)**：
  - 遇到长列表或庞大表单（如：新建线程承诺、新建日志、线程内部的修改表单、历史记录），必须放在 `<details>` 中默认折叠。
  - 对于带有警示状态的数据（如过期、失衡的线程），可以利用 `open` 属性默认展开。
- **表单与加载阻塞**：
  - 数据同步 (`Sync`) 或重算等高耗时请求，必须使用带全屏/局部遮罩的 `SubmitButton`。
  - 遮罩风格统一采用 `paper/80` 配合 `backdrop-blur`，中央展示无装饰的粗野提示块（如 `FETCHING LEDGER...`, `SYNCING...`）。
- **滚动条**：
  - 具有内部滚动需求的区域（如时间线 Timeline），需采用粗野主义风格滚动条 (`brutal-scrollbar`)，使用实色块而非圆角原生滚动条。

## 4. 路由与可见性 UX 边界

系统的路由设计应当使结构极其扁平，`/` 仅作为欢迎页，业务逻辑全部收敛于 `/dashboard`。

- **根路由 (`/`)**：
  - 纯静态欢迎页（Logo页）。
  - 无论什么身份，只展示 `AFLOAT V1` 标签、`浮生` 大标题与诗意文案：“不安排你的时间，只呈现承诺如何成为事实，或如何偏离事实。”
  - 唯一行动点：点击“进入镜像”前往 `/dashboard`。
- **工作台路由 (`/dashboard`)**：
  - **Owner（已登录）**：渲染完整版 `DashboardWorkbench`。
  - **Guest（未登录且访客模式开启）**：渲染隐藏了线程模块的访客版 `DashboardWorkbench`。
  - **Locked（未登录且访客模式关闭）**：显示 `SYSTEM LOCKED` / `ACCESS DENIED` 拦截界面，并提供通向 `/login` 的入口。
